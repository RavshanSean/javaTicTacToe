# Java Tic Tac Toe

This is a simple console-based Tic Tac Toe game implemented in Java.

## Features

- Two-player mode
- Displays the current game board
- Detects wins and draws

## How to Run

1. Ensure you have Java installed on your machine.
2. Clone this repository:
   ```bash
